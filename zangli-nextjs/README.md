# 百年藏历 - Next.js版本

基于Next.js的现代化藏历查询应用，提供1951年1月8日到2051年1月12日之间的公历藏历对照查询。

## ✨ 特性

- 📅 **100年藏历数据**：覆盖1951-2050年完整数据
- 🚀 **高性能**：按需加载数据，减少初始包大小
- 📱 **PWA支持**：支持离线使用，可安装到设备主屏幕
- ♿ **无障碍访问**：支持屏幕阅读器和键盘导航
- 📊 **响应式布局**：适配桌面和移动设备
- 🎨 **现代UI**：使用Tailwind CSS构建的美观界面

## 🛠️ 技术栈

- **框架**: Next.js 15 + React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS + CSS自定义属性
- **PWA**: Workbox Service Worker
- **数据**: 分离式JSON文件，按需加载

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 生成数据文件
```bash
node scripts/split-data.js
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000 查看应用

### 构建生产版本
```bash
npm run build
npm start
```

## 📊 数据结构

应用使用分离式数据文件：
- `src/data/[年份].json` - 单年数据文件
- `src/data/[年代]s.json` - 十年数据文件
- `src/data/index.json` - 数据索引文件

### 特殊日期标记

- **负数**: 缺日（当天不存在）
- **正数**: 闰日（当天重复）
- **0**: 闰月标记

## 🎯 使用说明

1. **选择日期**: 使用年月选择器或点击日历单元格
2. **查看藏历**: 选中日期后自动显示对应藏历信息
3. **导航**: 支持键盘快捷键
   - `←/→`: 上/下月
   - `↑/↓`: 上/下年
   - `T`: 回到今天
4. **节日信息**: 重要藏历节日会自动显示

## 📱 PWA功能

- **离线缓存**: 访问过的数据会缓存到本地
- **安装提示**: 可安装为原生应用
- **离线查询**: 支持无网络环境下查询

## 🎨 界面特色

- **CSS Grid布局**: 确保按钮在窄屏幕也能横排显示
- **语义化HTML**: 正确的table标签和ARIA属性
- **键盘导航**: 完整的键盘操作支持
- **屏幕阅读器**: 优化的无障碍访问体验
